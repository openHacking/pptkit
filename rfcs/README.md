# RFCs

This directory is for major design proposals and architecture decisions.

## Naming Convention

Use zero-padded identifiers such as:

- `0001-layout-engine.md`
- `0002-plugin-system.md`
- `0003-export-pipeline.md`

## When to Write an RFC

- New public extension points
- Major package boundary changes
- Breaking API shifts
- Significant data model or rendering architecture changes
