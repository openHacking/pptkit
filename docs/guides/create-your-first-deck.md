# Create Your First Deck

This guide will become the main practical walkthrough for new users once the exporter moves beyond the current placeholder result.

## Intended Scope

The first complete guide should cover:

1. Installing the required packages
2. Creating a presentation
3. Adding slides and elements
4. Running layout
5. Exporting a `.pptx`

## Documentation Goal

This guide should be the shortest path from zero to a working output file.

## Future Additions

After the first implementation lands, this guide should also link to:

- Text and typography guides
- Image and SVG guides
- Layout patterns
- Troubleshooting notes
