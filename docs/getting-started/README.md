# Getting Started

This section is the shortest path from an installed PPTKit package to editable PPTX output.

## Reading order

1. [Install PPTKit](installation.md) — install preview packages and choose package entry points.
2. [Quick Start](quick-start.md) — validate and write an editable PPTX.
3. [Hello World: Core Only](hello-world.md) — the smallest authoring-to-IR example.
4. [Create Your First Deck](../guides/create-your-first-deck.md) — layouts, assets, multiple slides, and error handling.

PPTKit preview packages are available on npm. User-facing examples target the current
preview package contract and intentionally omit superseded pre-release APIs.
